Ejercicios del alumno Jairo del Módulo Aplicaciones Web

Esta página contiene enlaces a los diferentes ejercicios que voy a realizar para el módulo LMSGE del IES Guadalpeña para el curso 2021-2022

Ejercicios

  • Página secundaria
  • Actividad 1. Prueba sobre como quedan visualmente todos los encabezados
  • Actividad 2. Prueba del elemento "p" con las etiquetas "center, left y right". Utiliza el elemento "pre" con las etiquetas "b" "i" y "tt" en diferentes sitios de la página
  • Actividad 3. Creación de una página donde se muestra la misma imagen con diferentes tamaños y utilizando la etiquetas "border" y "align"
  • Actividad 4. Creación de un listado NO enumerado de grupos musicales, otro listado enumerado de webs que visito regurlarmente y un listado de definiciones con los elementos "dl", "dt" y "dd"
